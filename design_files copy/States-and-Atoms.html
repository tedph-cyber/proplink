States, Atoms & Patterns

The shared building blocks every page reuses — buttons, prices, badges, form fields, status pills — and the easy-to-forget states: loading, empty, 404, and feedback. Build these as real components so the whole app stays consistent.

01ButtonsTwo variants + WhatsApp. Sizes sm/md/lg. WhatsApp button is sacred — never reskinned.
02Price & badgesAlways render price via the <Price> component. Verified chip is neutral — never green.
